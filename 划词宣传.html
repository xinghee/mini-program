快速上手

三步开始使用

从安装到出结果，不超过 2 分钟。

1

安装扩展

将项目文件夹加载到 Chrome（开发者模式 → 加载已解压的扩展），图标出现在工具栏。

2

配置 API

点击图标打开设置，填入你的 API 地址、密钥和模型名称（本地 Ollama 可不填密钥）。

3

划词即用

在任意网页选中文字，浮窗自动出现。点击翻译、总结、解释或朗读，结果即时显示。

深度定制

你的工具栏，你说了算

不只是预设按钮——自定义提示词、拖拽排序、朗读语音，打造专属划词工作流。

✏️

自定义按钮与提示词

翻译、总结只是开始。创建"改写为正式语气"、"提取关键词"、"对比分析"……提示词用 {{text}} 代表选中文字。

🔀

拖拽排序

把最常用的按钮拖到前面，不常用的往后放。设置页直接拖拽，即时生效。

🗣️

14 种语言智能朗读

自动检测中/英/日/韩/法/德/西/俄/葡/意/荷/波/印尼/印地语，匹配最佳语音。设置母语加速，母语内容以 1.5x~3.0x 播放，其他语言保持正常语速。

隐私优先

你的数据，只在你手里

🔒

零数据收集

不收集、不上传、不分析任何用户数据。API Key 存在浏览器本地 storage，AI 请求直接从你的浏览器发出。

🛡️

最小权限

仅申请 storage 权限保存设置。网络请求仅在用户主动划词时发出，API 域名权限由用户按需授权。
