# 单 HTML 应用

本目录包含一系列**单 HTML 文件**的小工具应用，无需安装、无需构建，直接在浏览器中打开即可使用。

## 文件列表

| 文件 | 说明 |
|------|------|
| `阅读.html` | 阅读工具 |
| `huaci.html` | 花词工具 |
| `check_voices .html` | 语音检查工具 |

## 使用方式

1. 下载对应的 `.html` 文件
2. 用浏览器（Chrome / Edge / Firefox 等）打开即可
3. 无需任何后端服务或依赖安装

## 技术栈

- 纯 HTML + CSS + JavaScript
- 零依赖，零构建
